**基于深度学习的图像分类与目标检测实验**

期末综合实践

#### 一、实验目的

本次上机作为计算机视觉课程的期末成绩考核，要求学生综合运用本课程所学的图像处理、特征提取、深度学习和视觉任务建模方法，完成一个完整的计算机视觉实践任务。

通过本次实验，学生应能够：

1.  理解计算机视觉任务的基本流程，包括数据准备、模型构建、训练、测试和结果分析；
2.  掌握至少一种深度学习视觉模型的使用方法；
3.  能够对实验结果进行定量评价和可视化分析；
4.  能够独立撰写实验报告，说明方法原理、实验过程、结果和问题分析；
5.  培养解决实际视觉问题的综合能力。

#### 二、实验内容

学生可从以下两个任务中任选一个完成。

##### 任务一：图像分类综合实验

1.  **任务描述**
    选择一个图像分类数据集，完成图像分类模型的训练、测试与分析。学生需要使用卷积神经网络或已有预训练模型，对图像进行类别识别，并对实验结果进行评价。

2.  **可选数据集**
    学生可选择以下数据集之一：
3.  CIFAR-10；
4.  MNIST / Fashion-MNIST；
5.  自行收集的小规模图像分类数据集；
6.  教师提供的课程数据集。

若学生自行收集数据集，要求：

- 不少于 3 个类别；
- 每个类别不少于 50 张图像；
- 需要划分训练集、验证集和测试集；
- 报告中说明数据来源与类别设置。

3.  **基本要求**
    学生需要完成以下内容：
4.  数据集读取与预处理；
5.  数据增强方法设计，例如翻转、裁剪、归一化等；
6.  构建或调用分类模型；
7.  完成模型训练；
8.  在测试集上进行分类预测；
9.  计算分类准确率；
10.  绘制训练损失曲线和准确率曲线；
11.  展示若干正确分类和错误分类样例；
12.  分析模型性能和错误原因。

13.  **可选模型**
     可使用以下模型之一：
14.  自定义 CNN；
15.  LeNet；
16.  AlexNet；
17.  VGG；
18.  ResNet；
19.  MobileNet；
20.  Vision Transformer 简化版本或预训练模型。

鼓励学生使用预训练模型并进行微调。

##### 任务二：目标检测综合实验

1.  **任务描述**
    选择一个目标检测数据集，完成目标检测模型的训练、测试与结果分析。学生需要使用目标检测模型对图像中的目标进行定位和分类，并输出检测框结果。

2.  **可选数据集**
    学生可选择以下数据集之一：
3.  Pascal VOC 子集；
4.  COCO 小规模子集；
5.  自行标注的小规模目标检测数据集；
6.  教师提供的目标检测数据集。

若学生自行构建数据集，要求：

- 至少包含 2 类目标；
- 总图像数量不少于 100 张；
- 每张图像需包含目标标注框；
- 可使用 LabelImg、Roboflow、CVAT 等工具进行标注。

#### 三、实验要求

1.  **程序要求**
    学生需要提交完整代码，代码应能够独立运行。代码中应包含必要注释，并保证文件结构清晰。

建议代码目录结构如下：

```
CV_Final_Project/
├── data/
├── models/
├── results/
├── train.py
├── test.py
├── visualize.py
├── requirements.txt
└── README.md
```

若使用 Jupyter Notebook，也需要保证代码运行顺序清晰，并附必要说明。

2.  **报告要求**
    实验报告应包含以下内容：
3.  实验题目；
4.  实验目的；
5.  数据集介绍；
6.  方法原理；
7.  实验环境；
8.  实验步骤；
9.  实验结果；
10.  可视化结果；
11.  结果分析；
12.  遇到的问题及解决方法；
13.  总结与体会。

报告中必须包含：

- 数据集样例图；
- 模型结构或方法流程说明；
- 训练损失曲线；
- 测试结果；
- 若干可视化预测结果；
- 对失败案例的分析。

压缩包中应包含：

```

├── 源代码/
```

其中：

1.  `源代码/`：包含训练、测试和可视化代码；

3.  `实验结果/`：保存预测结果图、训练曲线、评价结果等；
4.  `README.md`：说明如何运行代码；
5.  `requirements.txt`：列出所需 Python 库。
